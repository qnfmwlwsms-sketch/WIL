
# Mobile Wedding Invitation

## Overview

A simple, elegant, and mobile-friendly wedding invitation website. This project will use modern HTML, CSS, and JavaScript to create a beautiful user experience. It will be a single-page application.

## Design and Features

*   **Responsive Design:** The layout will be mobile-first, ensuring it looks great on all devices.
*   **Elegant Typography:** We will use custom fonts from Google Fonts to create a romantic and sophisticated feel.
*   **Color Palette:** A soft and romantic color scheme (e.g., blush pink, gold, cream).
*   **Interactive Photo Gallery:** A gallery of photos of the couple, implemented as a Web Component.
*   **Key Information:** Clear presentation of the bride and groom's names, wedding date, time, and location.
*   **Animations:** Subtle animations to enhance the user experience.
*   **Contact/RSVP:** A section with contact information.

## Current Plan

1.  **Structure (`index.html`):** Set up the basic HTML structure for the invitation, including sections for the main invitation, gallery, and contact information.
2.  **Styling (`style.css`):** Apply styles to create an elegant and responsive design. This includes typography, colors, and layout.
3.  **Functionality (`main.js`):** Create a custom element for the photo gallery.
